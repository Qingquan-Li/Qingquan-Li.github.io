# My Personal Website

- About Me
- My GitHub
- Travel Photos
- Resume
- Links: LinkedIn, Instagram...

## Tech Stack
  - React.js and Vue.js(For front-end rendering)
  - Bootstrap 5 (Cascading Style Sheets)
  - GitHub Pages (For Deployment)

## Deployment

- [GitHub Pages](https://create-react-app.dev/docs/deployment/#github-pages)

```bash
$ git checkout dev
$ npm run deploy
```

## Run locally

```bash
$ git checkout dev
$ npm start
```
